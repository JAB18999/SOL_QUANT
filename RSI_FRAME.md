# RSI Quantitative Analysis & Research Framework (Ultra v3.0)

**项目愿景**：通过对 RSI14 的深度结构化统计，解码市场在不同强弱状态下的持续性、盈亏期望及状态转换规律，建立一套完整的量化行为研究体系。

---

## 1. 基础架构设定

### 1.1 数据维度
- **标的**：SOL/USDT (及其他主流品种)
- **周期**：15分钟 (Short-term), 30分钟 (Intraday)
- **时区**：东八区 (UTC+8)
- **回溯窗口**：3d, 7d, 15d, 30d, 60d (滚动回测)

### 1.2 RSI 区间严格定义
| 区间名称 | 逻辑条件 | 状态语义 |
| :--- | :--- | :--- |
| **超卖区** | RSI14 ≤ 30 | 极度超跌 / 潜在反弹 |
| **中性偏空区** | 30 < RSI14 ≤ 50 | 弱势震荡 / 阴跌 |
| **中性偏多区** | 50 < RSI14 ≤ 70 | 强势震荡 / 慢牛 |
| **超买区** | RSI14 > 70 | 极度超买 / 潜在回调 |

## 2. 核心统计模块（含扩展）

### 2.1 基础分布与持续性统计
- **时间分布**：各区间 K 线数量占比。
- **事件频率**：各区间“进入-停留-离开”作为 1 个事件的发生频次。
- **生命周期**：
  - 单次事件平均持续 K 线数、最大持续 K 线数。
  - **衰减分析**：统计事件持续超过 N 根 K 线后的概率分布（用于判断趋势衰竭）。

### 2.2 价格行为与波动特征
- **单 K 线涨跌幅**：均值、标准差（波动率）、偏度。
- **区间收益率 (Event Return)**：进入区间到离开区间的总变动。
- **成交量加权统计 (VWAR)**：
  - 计算各区间内的平均成交量。
  - **成交量强度因子** = 区间平均成交量 / 总样本平均成交量。
  - *用途：验证超买/超卖区是否伴随放量，识别真假突破。*

### 2.3 风险与极限分析 (MFE/MAE)
对每一个区间事件进行过程监控：
- **MFE (最大有利偏移)**：区间内价格达到的最高（多头）/最低（空头）获利点。
- **MAE (最大不利偏移)**：区间内价格出现的最大回撤点。
- **收益风险比 (Efficiency Ratio)**：Average MFE / Average MAE。

## 3. 高级扩展模块（Research Deep-Dive）

### 3.1 RSI 状态转移概率矩阵 (Markov Chain)
统计 RSI 从一个区间切换到另一个区间的“条件概率”，生成 4x4 转移矩阵：
- **计算逻辑**：P(当前区间 | 前一区间)。
- **研究目标**：
  - 识别“状态粘性”：哪个区间最容易产生长时间停留？
  - 识别“路径偏好”：超买后是回落到中性偏多，还是直接跳空杀入超卖？

### 3.2 RSI 指标背离统计 (Divergence Analysis)
统计价格与 RSI 走势的不一致性：
- **牛市背离 (Bullish Divergence)**：价格创近期新低，但 RSI14 处于超卖区且低点抬升。
- **熊市背离 (Bearish Divergence)**：价格创近期新高，但 RSI14 处于超买区且高点下降。
- **统计字段**：背离发生后的 N 根 K 线反转成功率、平均涨跌幅。

### 3.3 多时间框架共振统计 (MTF Resonance)
- **统计规则**：
  - 当 30m 处于“超买区”时，15m 处于各区间的分布情况。
  - 当 15m 与 30m 同时处于“超买”或“超卖”时的加速效应分析。
- **研究目标**：寻找高概率的“嵌套式”入场点。

### 3.4 后续期望收益统计 (Forward Expectancy)
统计离开某区间后的未来表现：
- 离开该区间后，未来 1、3、5、10 根 K 线的价格变动均值。
- **胜率统计**：离开超卖区后，未来 N 根 K 线价格高于当前价格的概率。

## 5. 预期研究成果与应用

1. **基准概率库**：建立 SOL 在不同周期下的 RSI 行为常数（如：超买通常持续多久？）。
2. **过滤信号**：通过 MAE 分析设定科学的止损位，通过 MFE 设定止盈位。
3. **策略雏形**：
   - **均值回归**：利用背离统计和超买/卖反转概率。
   - **趋势追踪**：利用中性偏多区间的持续性特征。
4. **异常预警**：当当前 RSI 持续时间超过历史 95% 分位数时，发出衰竭预警。
