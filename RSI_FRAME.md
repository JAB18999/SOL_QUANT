# RSI Statistical Analysis Framework (RSI_FRAME)

**项目目标**：对 SOL（或其他标的）在 15分钟 和 30分钟 K线上进行 RSI14 的结构化量化统计分析，构建可重复、可验证的 RSI 行为数据库，为后续策略开发提供坚实的数据支撑。

**适用周期**：15分钟K线、30分钟K线  
**时区**：统一使用东八区（北京时间）

---

## 1. 基础设定

### 1.1 时间跨度
统计以下多个时间窗口（分别独立计算）：
- 最近 60 天
- 最近 30 天
- 最近 15 天
- 最近 7 天
- 最近 3 天

### 1.2 RSI 参数
- RSI 周期：14（RSI14）
- 计算依据：每根K线的**收盘价**
- 中轴值：50
- 经典阈值：30（超卖）、70（超买）

### 1.3 RSI 区间划分（严格全覆盖、无重叠）

| 区间名称     | 条件                  | 说明           |
|--------------|-----------------------|----------------|
| 超卖区       | RSI14 ≤ 30            | 强弱势状态     |
| 中性偏空区   | 30 < RSI14 ≤ 50       | 弱势震荡       |
| 中性偏多区   | 50 < RSI14 ≤ 70       | 强势震荡       |
| 超买区       | RSI14 > 70            | 强强势状态     |

**重要原则**：每个 RSI14 值必须且只能属于一个区间。

---
